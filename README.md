How It Works
User Authentication: Users log in with their university credentials via Clerk.

Role-Based Access:

Admins can manage the full inventory, approve requests, and modify stock.

Users (e.g., students, faculty) can view and request inventory.

Inventory Management:

Users can see real-time stock levels.

Admins can update stock, add new items, and track issued components.

Request & Approval Flow:

Users request items they need.

Admins receive, approve, or reject requests in a dashboard.

Issued items are tracked until they are returned.
